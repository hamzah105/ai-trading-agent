About The Agent

Hamza's AI Trading Agent combines cutting‑edge machine learning with traditional technical analysis to identify high‑probability trade setups. Designed to be modular, secure, and beginner‑friendly, the system can be run locally in simulation (“paper”) mode or connected to a live exchange with proper API keys.

  • ✅ Real‑time market data ingestion
  • ✅ ML‑driven price predictions
  • ✅ Configurable risk parameters
  • ✅ Built‑in backtesting engine
  • ✅ Fully open‑source and customizable

Features

🧠 AI Model

Uses advanced time‑series models to forecast price direction and volatility.

🔒 Risk Management

Automatic stop‑loss, take‑profit, and position sizing rules protect your capital.

📈 Backtesting

Test strategies against years of historical data before risking real money.
